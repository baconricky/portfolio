RED HAT STORAGE ARCHITECTURE

Red Hat Storage Server is built using the GlusterFS open source project as the foundation. GlusterFS is a distributed file system that can scale to several petabytes, runs on commodity hardware, and aggregates storage capacity and computing resources from various servers.

Discover unique technical aspects of GlusterFS including:

  • Eliminate performance bottlenecks with a no metadata server architecture
  • Achieve linear scalability and high availability
  • Maximize performance via a fully distributed architecture

To start a functional trial of Red Hat Storage Server on GlusterFS, visit RedHat.com/Liberate.
